# Tic Tac Toe (Python)

This project is a console-based Tic Tac Toe game developed as part of a Python course.

## Overview

The program allows a user to play Tic Tac Toe against a simple computer opponent. The computer uses random moves instead of artificial intelligence.

## Game Rules

* The computer plays as **'X'** and always starts in the center of the board
* The player uses **'O'**
* The board positions are numbered from 1 to 9
* The player selects a move by entering a number
* Moves must be valid and cannot overwrite occupied spaces

## Features

* Input validation and error handling
* Turn-based gameplay
* Win detection (rows, columns, diagonals)
* Draw detection when no moves remain
* Randomized computer moves

## Technologies Used

* Python (standard library)
* `random` module
* `time` module

## Purpose

This project was created as a learning exercise to practice:

* Control flow (loops and conditionals)
* Functions and modular programming
* Data structures (lists and tuples)
* Basic game logic implementation
